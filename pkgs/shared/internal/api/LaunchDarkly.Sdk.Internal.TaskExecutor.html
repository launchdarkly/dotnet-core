    TaskExecutor(object, Action<Action>, Logger)

    Creates an instance, specifying custom event dispatch logic.

    Declaration
    public TaskExecutor(object eventSender, Action<Action> eventHandlerDispatcher, Logger log)
    Parameters
    Type Name Description
    object eventSender

    object to use as the sender parameter when firing events
    Action<Action> eventHandlerDispatcher

    custom logic to use when executing an event handler, or null to use the default behavior
    Logger log

    logger for logging errors from worker tasks
    Remarks

    The eventHandlerDispatcher parameter, if not null, specifies what to do when calling a lambda that executes an application-defined event handler. The default behavior for this is that we call Task.Run for each handler invocation to run it as a separate background task. In the client-side .NET SDK, we may want to use a different approach (such as invoking the handler on the UI thread, for mobile devices).
